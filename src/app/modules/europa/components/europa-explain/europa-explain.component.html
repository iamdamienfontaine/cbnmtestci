Programmes de lutte contre les espèces végétales exotiques envahissantes sur l’île Europa

Sur l’île Europa (30 km2), les programmes de lutte concernent neuf espèces végétales exotiques envahissantes : le Cenchre épineux (Cenchrus echinatus), la Fataque (Urochloa maxima), le Choca (Furcraea foetida), le Sisal (Agave sisalana), le Ricin (Ricinus communis), le Filao (Casuarina equisetifolia), le Tamarin (Tamarindus indica), le Morongue (Moringa oleifera) et le Cocotier (Cocos nucifera). À ce jour, seules les actions entreprises contre le Choca, le Sisal et la Fataque sont présentées sur ce site WEB (compléments envisagés courant 2021)

Conçu sous la forme d’une interface cartographique, ce site permet de connaitre la localisation des populations d’EEE, de consulter en détail les diverses actions (lutte ou suivi) menées contre les EEE et de dresser des bilans (à l’échelle de chaque population ou à l’échelle de chaque EEE). Ces données sont régulièrement actualisées et complétées au gré des actions menées in situ.

Règles de fonctionnement

Bandeau 'Populations'

  • Cliquer sur l’icône chrome_reader_mode (en haut à gauche sur les terminaux mobiles) afin de faire apparaitre le bandeau ‘Populations’.

  • Cliquer sur le bouton afin de consulter le bilan global des actions de lutte et de suivi menées sur chaque taxon.
    Sélectionner le sous-onglet correspondant au taxon recherché : ‘Furcraea foetida (Choca)’ ou ‘Agave sisalana (Sisal)’ ou ‘Urochloa maxima (Fataque)’.

  • Sélectionner le taxon dont les populations doivent être affichées sur la carte au niveau du menu déroulant du bandeau ‘Populations’ : ‘Furcraea foetida - Choca’ ou ‘Agave sisalana - Sisal’ ou ‘Urochloa maxima - Fataque’.
    Les populations apparaissent sur la carte (Cf. ‘Carte') ainsi que la liste des populations (n° de population ; état de la lutte ; nombre total d’actions de lutte / de suivi).

  • Cliquer sur le n° de population souhaité (ou sur la population affichée sur la carte) pour faire apparaitre une nouvelle fenêtre d’affichage des informations associées à la population sélectionnée.
    Sélectionner le sous-onglet correspondant aux informations recherchées : ’Etat initial’ ou ‘Actions de lutte/suivi’ ou ‘Bilan des actions’.
    Dans le cas du sous-onglet ‘Actions de lutte/suivi’, il faut d’abord choisir le type d’action (‘Lutte’ ou ‘Suivi’) dans le menu déroulant, puis cliquer sur une date (classement par ordre chronologique décroissant) afin d’obtenir toutes les informations relatives à l’action sélectionnée.

Carte

  • La carte affiche les populations de l’EEE sélectionnée au niveau du menu déroulant du bandeau ‘Populations’. Il y a possibilité de se déplacer sur la carte, de zoomer ou dézoomer.

  • Cliquer sur l’icône layers pour faire apparaitre les légendes.

  • Cliquer sur une population affichée sur la carte (ou sur le n° de population au niveau du bandeau ‘Populations’) pour faire apparaitre une nouvelle fenêtre d’affichage des informations associées à la population sélectionnée.
    Sélectionner le sous-onglet correspondant aux informations recherchées : ’Etat initial’ ou ‘Actions de lutte/suivi’ ou ‘Bilan des actions’.
    Dans le cas du sous-onglet ‘Actions de lutte/suivi’, il faut d’abord choisir le type d’action (‘Lutte’ ou ‘Suivi’) dans le menu déroulant, puis cliquer sur une date (classement par ordre chronologique décroissant) afin d’obtenir toutes les informations relatives à l’action sélectionnée.
